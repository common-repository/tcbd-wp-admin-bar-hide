= Plugin Features =

* Easy to use.
* Supper Light (Only 1KB).


== Installation ==

1. Upload 'tcbd-admin-bar-hide' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
